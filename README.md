 # 📊 Northwind Sales Analysis Project

This project analyzes the **Northwind sales dataset** using **MySQL and Power BI** to uncover insights about sales performance, customer behavior, and product trends.

The goal of this project is to demonstrate **data analysis skills using SQL and data visualization using Power BI**.

---

## 🛠 Tools & Technologies

* MySQL Workbench
* SQL
* Power BI
* Northwind Dataset

---

## 📁 Project Structure

northwind-sales-analysis

├── SQL → SQL queries used for data analysis
├── PowerBI_Dashboard → Interactive Power BI dashboard
├── Screenshots → Dashboard preview images
└── README.md → Project documentation

---

## 📊 SQL Analysis

The SQL analysis includes:

* Creating a **sales_fact view** by joining multiple tables
* Revenue analysis
* Customer contribution analysis
* Product performance analysis
* Monthly revenue trends
* Customer retention and cohort analysis

---

## 📈 Dashboard Insights

The Power BI dashboard contains four main sections:

### 1️⃣ Overview

* Total Revenue
* Total Orders
* Average Order Value
* Total Customers
* Monthly Revenue Trend

### 2️⃣ Sales Performance

* Revenue vs Orders Trend
* Salesperson Performance
* Top Products by Revenue

### 3️⃣ Customer Insights

* Top Customers by Revenue
* Customer Order Volume
* Revenue Share by Customer

### 4️⃣ Product Performance

* Top Products by Revenue
* Product Quantity Sold
* Revenue vs Quantity Analysis

---

## 🎯 Key Insights

* A small number of customers contribute a large portion of revenue.
* Some products generate significantly higher revenue than others.
* Sales fluctuate across months indicating seasonal patterns.
* Salesperson performance varies across the dataset.
